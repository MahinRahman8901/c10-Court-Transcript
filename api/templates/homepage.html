Welcome to the Courtroom API

The Courtroom API provides access to information about court cases, judges, and circuits. It allows you to retrieve data programmatically for various use cases.

How to Use the API

  • /cases: Retrieves a list of all court cases.

    This endpoint returns a list of all court cases available in the database.

    Filters available:

    • judge_id: Filter cases by judge ID.
    • search: Filter cases by search query.
  • /cases/{case_no}: Retrieves information about a specific court case identified by its case number.

    This endpoint retrieves detailed information about a specific court case based on its unique case number.

  • /judges: Retrieves information about all the judges.

    This endpoint provides information about all the judges presiding over the court cases.

    Filters available:

    • circuit_id: Filter judges by circuit ID.
    • judge_type_id: Filter judges by judge type ID.
  • /judges/{judge_id}: Retrieves information about a specific judge identified by their ID.

    This endpoint retrieves detailed information about a specific judge based on their unique ID.

  • /judge_types Retrieves information about each judge type.

    This endpoint retrieves detailed information each different judge type available.

  • /circuits: Retrieves a list of all circuits.

    This endpoint returns a list of all circuits associated with the court cases.

You can use these endpoints to integrate court case data into your applications, analyze legal trends, or build tools for legal professionals.
